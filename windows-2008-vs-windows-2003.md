# Windows 2008 还是 Windows 2003

## Windows 2008

*  优点：目前最主流的Windows系统，功能强大
*  缺点：相比Windows 2003变化较大，熟悉Windows 2003的用户会比较不习惯。资源消耗稍多。

推荐多数用户使用Windows 2008。

## Windows 2003

*  优点：有很多熟悉Windows 2003的用户
*  缺点：半退休状态，即将与2015年完全退休。

除非软件兼容性问题不得已，不建议使用。
